##Emergency Department Utilization & 2047 Capacity Forecast
*Project Overview
This project analyzes historical hospital data to evaluate current Emergency Department (ED) performance and forecast future bed capacity requirements through the year 2047. It serves as a strategic tool for hospital executives to make data-driven decisions regarding facility expansion and resource allocation.
*Key Features & Insights
Capacity Forecasting: Developed predictive models to estimate hospital demand and utilization rates up to 2047.
Operational Metrics: Tracked real-time performance indicators, including patient inflow and discharge efficiency.
Interactive Visualizations: Designed intuitive dashboards for stakeholders to filter data by department and timeframes.
*Technical Toolkit
Power BI: Data modeling and interactive dashboard design.
DAX (Data Analysis Expressions): Created complex measures for utilization rates and growth projections.
Power Query: Performed data transformation and cleaning to ensure reporting accuracy.
